# 💰 Personal Expense Tracker

A simple and interactive web-based Expense Tracker built using Python and Streamlit. This application helps users track daily expenses, analyze spending patterns, and manage finances efficiently.

---

## 🚀 Live Demo
👉  https://expense-tracker-ngfmvf4qynchrscapagxbk.streamlit.app/

---

## 📌 Features

- Add daily expenses with date, amount, category, and description  
- View all expenses in a structured table  
- Category-wise spending analysis  
- Summary including Total, Average, and Maximum expenses  
- Filter expenses by date and category  
- Download expense data as CSV file  

---

## 🛠️ Technologies Used

- Python  
- Streamlit  
- Pandas  
- Matplotlib

## ⚙️ How to Run Locally

1. Clone the repository  
git clone https://github.com/abiramasree/expense-tracker.git  

2. Go to project folder  
cd expense-tracker  

3. Install dependencies  
pip install -r requirements.txt  

4. Run the application  
streamlit run app.py
